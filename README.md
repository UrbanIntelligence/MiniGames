# MiniGames

A tiny browser arcade with two games on one page:
- Snake
- Breakout

## Local run

```bash
python3 -m http.server 4173
```

Open [http://localhost:4173](http://localhost:4173).

## Controls

- Switch game using `Snake` / `Breakout` buttons.
- `Space`: pause/resume active game
- `R`: restart active game
- Snake: Arrow keys or WASD
- Breakout: Left/Right arrows or A/D

## Public site

This repository is configured to deploy to GitHub Pages via GitHub Actions.
After the first push, check the **Actions** tab for deployment status.
